# Interview Scheduler

## Setup

Install dependencies with `npm install`.

### Dependencies
* axios
* classnames
* fsevents
* normalize.css
* react
* react-dom
* react-scripts

## Running Webpack Development Server

```sh
npm start
```

## Running Jest Test Framework

```sh
npm test
```

## Running Storybook Visual Testbed

```sh
npm run storybook
```
